This is a Tic-Tac-Toe game built with React, based on the official React tutorial.

## Features
- 🎮 **Dynamic Grid**: Supports 3x3, 4x4, 5x5, and even 10x10 boards.
- 📍 **Move History**: Tracks coordinates (row, col) for every turn.
- 🏆 **Smart Win Logic**: Dynamic algorithm to calculate winners on any board size.
- 🎨 **UI Enhancements**: Highlights the winning combination and supports history sorting.
